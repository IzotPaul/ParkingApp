# ParkingApp

Usage:

1. Open the server:
    python3 server.py
    
2. Open the map_creator and create an amazing map. Hit SAVE! after finishing:
    python3 map_creator.py
    
3. From the folder client/ open as many clients as desired:
    python3 client.py
    
    
    
Libraries needed:
  numpy
  peeweee
  opencv-python
  scypi
  scikit-learn
